产品购买份数查询

产品购买份数查询 | /product/info/buyNum

接口说明

该接口用于营销活动根据产品ID查询显示出产品购买份数
调用地址 /product/info/buyNum
授权类型 对外开放API
请求方法 GET
请求参数
参数 必选 说明
productId Y 产品ID(多个以逗号隔开)
