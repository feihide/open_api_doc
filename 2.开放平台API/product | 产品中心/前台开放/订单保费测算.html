订单保费测算

订单保费测算 | /product/rate/calculation

接口说明

该接口包括测算和下订单，当isOrder=0时，表示保费测算；isOrder=1时，表示下订单。
调用地址 /product/rate/calculation
授权类型 前台开放
请求方法 POST
请求参数
参数 必选 说明
appntID N 投保人ID（如果保费测算，则传空）
productID Y 产品ID
productName N 产品名称
effDate N 生效日期
appntIDType N 投保人证件类型
appntIDNo N 投保人证件号
homeAddress N 投保人家庭住址
province N 投保人住址省级
city N 投保人住址市级
area N 投保人住址区级
bnfType N 受益人类型（如果是保费测算，传空值。订单为必传）
bnfName N 受益人名称（如果是保费测算，传空值。订单为必传）
bnfIDType N 受益人证件类型
bnfIDNo N 受益人证件号
relationToInsured N 受益人与投保人关系
isOrder Y 是否下订单（0 代表保费测算，1 代表下订单）
insuredsStr Y 被保人信息，Json字符串，包含所选险种信息（格式为：[{ "insuredID":"", "name":"", "relationType":"", "certType":"", "certNo":"", "birthday":"2008-11-23", "sex":"0", "risks":[{ "planCode":"4", "planPeriod":"12个月", "preAmount
email N 投保人邮箱
appntName N 投保人姓名
mobile N 手机号
openBank N 开户行
bankCard N 卡号
bankOwner N 开户名
